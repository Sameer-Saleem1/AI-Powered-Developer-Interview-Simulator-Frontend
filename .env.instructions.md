# Environment Variables Setup

## Local Development

The `.env` file contains your environment variables for local development.

## Production (Vercel)

Add the following environment variable in your Vercel project settings:

**Key:** `VITE_API_URL`  
**Value:** `https://ai-powered-developerinterview-backend.netlify.app`

### How to add in Vercel:

1. Go to your Vercel project dashboard
2. Navigate to Settings â†’ Environment Variables
3. Add the variable above
4. Redeploy your project
